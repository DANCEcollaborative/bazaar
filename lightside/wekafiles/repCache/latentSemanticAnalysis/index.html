latentSemanticAnalysis: Performs latent semantic analysis and transformation of the data

URL:http://weka.sourceforge.net/doc.packages/latentSemanticAnalysis
Author:Amri Napolitano <amrifau{[at]}gmail.com>
Maintainer:Amri Napolitano <amrifau{[at]}gmail.com>

Performs latent semantic analysis and transformation of the data. Use in conjunction with a Ranker search. A low-rank approximation of the full data is found by specifying the number of singular values to use.
