regressionByDiscretization: A regression scheme that employs any classifier on a copy of the data that has the class attribute (equal-width) discretized.

URL:http://weka.sourceforge.net/doc.packages/regressionByDiscretization
Author:Eibe Frank and Len Trigg
Maintainer:Weka team <wekalist{[at]}list.scms.waikato.ac.nz>

A regression scheme that employs any classifier on a copy of the data that has the class attribute (equal-width) discretized. The predicted value is the expected value of the mean class value for each discretized interval (based on the predicted probabilities for each interval).

This package moved back into the core Weka distribution in Weka 3.7.3.
