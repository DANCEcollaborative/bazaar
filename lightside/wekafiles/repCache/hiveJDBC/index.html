hiveJDBC: A package containing the JDBC driver and dependencies for the Apache Hive database, along with a DatabaseUtils.props file for use with Weka.

URL:https://github.com/pentaho/hive
Author:Mark Hall
Maintainer:Mark Hall <mhall{[at]}pentaho.com>

A package containing the JDBC driver and dependencies for the Apache Hive database, along with a DatabaseUtils.props file for use with Weka. The driver supports reading from Hive but not writing.
