classificationViaRegression: Class for doing classification using regression methods.

URL:http://weka.sourceforge.net/doc.packages/classificationViaRegression
Author:Eibe Frank and Len Trigg
Maintainer:Weka team <wekalist{[at]}list.scms.waikato.ac.nz>

Class for doing classification using regression methods. Class is binarized and one regression model is built for each class value. For more information, see, for example

E. Frank, Y. Wang, S. Inglis, G. Holmes, I.H. Witten (1998). Using model trees for classification. Machine Learning. 32(1):63-76.

This package moved back into the core Weka distribution in Weka 3.7.3.
