Category:Classification

Depends:weka (=3.7.2)
Description:Class for doing classification using regression methods. Class is binarized and one regression model is built for each class value. For more information, see, for example

E. Frank, Y. Wang, S. Inglis, G. Holmes, I.H. Witten (1998). Using model trees for classification. Machine Learning. 32(1):63-76.

This package moved back into the core Weka distribution in Weka 3.7.3.
