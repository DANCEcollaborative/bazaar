arxAnonymizer: ARX Anonymization Filter

URL:https://github.com/farbtrommel/arx-weka
Author:Christian Windolf <christianwindolf{[at]}web.de>, Simon Könnecke <simon.koennecke{[at]}fu-berlin.de>, Andre Breitenfeld <andre.breitenfeld{[at]}fu-berlin.de>
Maintainer:Christian Windolf <christianwindolf{[at]}web.de>, Simon Könnecke <simon.koennecke{[at]}fu-berlin.de>, Andre Breitenfeld <andre.breitenfeld{[at]}fu-berlin.de>

This package provides algorithms to apply k-Anonymity, l-Diversity and t-Closeness with traversal strategy "Flash", for an optimal transformation to an anonymized dataset with respect to minimal information loss. The algorithms originate from the ARX Data Anonymization Toolkit, please read for further information's: http://arx.deidentifier.org/development/algorithms/.
