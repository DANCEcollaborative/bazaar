Author:Christian Lamberty <clamber{[at]}students.uni-mainz.de>

This package provides a regular vine model using sequential structure selection with pairwise maximum likelihood estimation as copula fitting method. This work was supported by a fellowship within the FITweltweit programme of the German Academic Exchange Service (DAAD).
