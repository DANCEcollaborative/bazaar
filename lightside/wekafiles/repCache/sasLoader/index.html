sasLoader: SAS sas7bdat file reader

URL:http://sassyreader.eobjects.org/
Author:Mark Hall <mhall{[at]}pentaho.com>
Maintainer:Mark Hall <mhall{[at]}pentaho.com>

Provides a loader for SAS's sas7bdat file format. Uses the eobjects.org "SassyReader" project. Sample sas7bdat data can be found in ${WEKA_HOME}/packages/sasLoader/sample-data.
