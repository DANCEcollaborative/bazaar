Description:Text filters for converting tweets into feature vectors for sentiment analysis. Sample tweets annotated by sentiment and emotions can be found in ${WEKA_HOME}/packages/AffectiveTweets/data. A file with some pre-trained word embeddings can be found in ${WEKA_HOME}/packages/AffectiveTweets/resources/w2v.twitter.edinburgh.100d.csv.gz. However, we recommend using the embeddings located in: https://github.com/felipebravom/AffectiveTweets/releases/download/1.0.0/w2v.twitter.edinburgh10M.400d.csv.gz for better performance.
