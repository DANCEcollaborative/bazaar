metaphorSearchMethods: An implementation of metaphor search methods to explore the space of attributes.

Author:DACC <mb45467{[at]}connect.umac.mo>, Silvia CHEN <yeding525{[at]}gmail.com>
Maintainer:DACC <mb45467{[at]}connect.umac.mo>, Silvia CHEN <yeding525{[at]}gmail.com>

Metaphor Search explores the attribute space using the metaphor search methods. The package includes AntSearch, BatSearch, BeeSearch, CuckooSearch, ElephantSearch, FireFlySearch, FlowerSearch, HarmonySearch and WolfSearch.
