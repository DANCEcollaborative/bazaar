Category:Clustering
Date:2014-06-02
Depends:weka (>=3.7.1)
Description:The package contains a new clustering algorithm based on MTree, witch also uses EM to determine when and how a node will be split. The main improvement this new clusterer brings is the possibility to use MTree specific methods like RangeQuery and KNN that are theoretically superior to those already implemented for weka clusterers. The package also contains a Test class.
