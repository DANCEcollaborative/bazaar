extraTrees: Package for generating a single Extra-Tree

URL:http://weka.sourceforge.net/doc.packages/extraTrees
Author:Eibe Frank
Maintainer:Weka team <wekalist{[at]}list.scms.waikato.ac.nz>

Package for generating a single Extra-Tree. Use with the RandomCommittee meta classifier to generate an Extra-Trees forest for classification or regression. This classifier requires all predictors to be numeric. Missing values are not allowed. Instance weights are taken into account. For more information, see

Pierre Geurts, Damien Ernst, Louis Wehenkel (2006). Extremely randomized trees. Machine Learning. 63(1):3-42.
