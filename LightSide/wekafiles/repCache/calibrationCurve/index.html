calibrationCurve: VisualizePlugin for plotting class probability calibration curves

URL:http://weka.sourceforge.net/doc.packages/calibrationCurve
Author:Eibe Frank
Maintainer:Weka team <wekalist{[at]}list.scms.waikato.ac.nz>

VisualizePlugin that can be used in the Classify panel of the Explorer or the Knowledge Flow to plot class probability calibration curves. Accessible under Plugins when right-clicking on the corresponding entry in the Result list of the Classify panel. The number of bins for equal-frequency discretization of the estimated class probabilities is chosen as the square root of the number of instances.
