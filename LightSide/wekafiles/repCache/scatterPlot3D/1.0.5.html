MessageToDisplayOnInstallation:This package includes Java3D classes and native
32 and 64 bit libraries for Windows, Linux and Mac OSX. No special installation is
required and the correct native libraries should be loaded automatically. The only
exception to this is Mac OSX when using Oracle Java 1.7 - in this case, if Apple's
Java 1.6 JVM is (or has been) installed, there will be an old version of Java3D
jar files and native libraries in /System/Java/Extensions. These will need to be
removed or they will interfere with the new version included in this package.
Note that, as of writing, there is a positioning bug when running under Oracle
Java 1.7 under Mac OSX. This results in the display being offset slightly in the
Window. It can be rectified (temporarily) by resizing the Explorer/Knowledge Flow
Window.
