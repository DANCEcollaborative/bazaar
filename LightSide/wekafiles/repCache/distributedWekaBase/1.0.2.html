Changes:ARFF header task now has options to allow a default label to be specified for a nominal attribute. This allows labels to be partially user-specified for a given nominal attribute, with all unspecified ones getting assigned to the default label. One use case is to reduce the number of classes in a multi-class problem by collapsing some into the default label.
