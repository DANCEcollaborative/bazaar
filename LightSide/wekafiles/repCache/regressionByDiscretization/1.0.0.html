Author:Eibe Frank and Len Trigg
Category:Regression
Date:2009-09-10
Depends:weka (=3.7.2)
Description:A regression scheme that employs any classifier on a copy of the data that has the class attribute (equal-width) discretized. The predicted value is the expected value of the mean class value for each discretized interval (based on the predicted probabilities for each interval).

This package moved back into the core Weka distribution in Weka 3.7.3.
