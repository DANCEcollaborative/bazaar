OpenmlWeka: Openml Weka

URL:https://github.com/openml/openml-weka
Author:Jan N. van Rijn <j.n.van.rijn{[at]}liacs.leidenuniv.nl>
Maintainer:Jan N. van Rijn <j.n.van.rijn{[at]}liacs.leidenuniv.nl>

Allows sharing Weka experiments on OpenML. Stores for every experiment the classifier, the (serialized) models, the predictions and evaluation measures. Works well in combination with the openml apiconnector (available on Maven Central, artifact id org.openml.apiconnector). If you found this package useful, please cite: J. N. van Rijn, Massively Collaborative Machine Learning, Leiden University, 2016.
