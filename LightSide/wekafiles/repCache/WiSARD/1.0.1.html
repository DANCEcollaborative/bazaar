Description:A multi-class classification method based on the WiSARD weightless neural model.

A data-preprocessing filter allows to exploit WiSARD neural model training/classification capabilities on multi-attribute numeric data, whereas WiSARD's was historically restricted to binary pattern recognition.

For more information, see

De Gregorio M, Giordano, M. (2018). An experimental evaluation of weightless neural networks for multi-class classification. Applied Soft Computing. Vol.72.
