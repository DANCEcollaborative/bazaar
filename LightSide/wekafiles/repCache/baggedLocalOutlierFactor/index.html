baggedLocalOutlierFactor: Filter implementing the Bagged LOF outlier/anomaly detection algorithm.

URL:http://people.cs.umass.edu/~lfriedl/code/BaggedLOF.html
Author:Lisa Friedland <lisadfriedland{[at]}gmail.com>
Maintainer:Lisa Friedland <lisadfriedland{[at]}gmail.com>

A filter that applies the Bagged LOF (Local Outlier Factor) algorithm to compute an "outlier" score for each instance in the data.

For more information, see :

Aleksandar Lazarevic, Vipin Kumar (2005). Feature Bagging for Outlier Detection. Proceedings of the Eleventh ACM SIGKDD International Conference on Knowledge Discovery in Data Mining, pp. 157-166.
