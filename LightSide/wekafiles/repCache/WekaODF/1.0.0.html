Description:WekaODF adds support to directory read from and write to spreadsheets in ODF (Open Document Format for Office Applications, ISO/IEC 26300:2006) format. ODF is used by the OpenOffice.org suite, for instance. WekaODF uses ODFDOM (http://odftoolkit.org/projects/odfdom, Apache 2.0) in order to read/write ODF spreadsheets.
