Author:Mark Hall
Category:Attribute selection
Date:2009-09-10
Depends:weka (>=3.7.1)
Description:This package provides two meta attribute selection evaluators that can apply an arbitrary filter to the input data before executing the actual attribute selection scheme. One filters data and then passes it to an attribute evaluator (FilteredAttributeEval), and the other filters data and then passes it to a subset evaluator (FilteredSubsetEval).
