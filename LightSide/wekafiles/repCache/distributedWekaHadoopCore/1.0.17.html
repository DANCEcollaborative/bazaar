Description:Provides loaders and savers for HDFS, plus Hadoop jobs and tasks that wrap the tasks provided in distributedWekaBase. This is the core of the distributedWekaHadoop package. To install all necessary componets for distributedWekaHadoop install either the distributedWekaHadoop or distributedWekaHadoop2 package, for Hadoop 1.x or 2.x respectively. This will ensure that the core package gets installed along with the appropriate libs package for either Hadoop 1.x or 2.x.
