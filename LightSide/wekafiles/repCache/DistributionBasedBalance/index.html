DistributionBasedBalance: Distribution-based balancing of datasets

Author:Pablo Bermejo<Pablo.Bermejo{[at]}uclm.es>
Maintainer:Pablo Bermejo <Pablo.Bermejo{[at]}uclm.es>

It learns one of the four available distributions from a set of instances, and replaces such a set resampling, for each class value, a given number of instances from the learnt distribution.
